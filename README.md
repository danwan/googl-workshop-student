# 🎓 TechBond Agent Academy: Student materials

Use this page to start the workshop, choose today's route, and open any lab in the complete library. Each lab contains the instructions, hidden hints, a collapsible solution route, and a success check.

## Start your workshop

- **[Day 1](./DAY_1.md):** NotebookLM, Agent Designer, and the fixed Antigravity taster
- **[Day 2](./DAY_2.md):** shared newcomer start, Cloud, ADK, Antigravity, and tools

The day pages are the primary workshop routes. They tell you what to complete together, what to choose, and what to clean up.

## The TechBond story

You've joined **TechBond Industries**, a fictional company that makes industrial adhesives for cars, electronics, and factories. TechBond has product sheets, customer emails, sales reports, quality incidents, and packed calendars. In each lab, you solve one of those business problems with AI.

All workshop emails, events, and documents are set in January 2026.

## Before you start

1. **Use your assigned workshop account and project:** never use another participant's project.
2. **Complete [Lab 0.1](./0-setup/lab-0.1-environment-setup.md):** create your Gemini Enterprise app and connect the workshop data. Lab 0.2 is helpful for Cloud work, but it is not an ADK prerequisite.
3. **Open NotebookLM:** sign in at [NotebookLM](https://notebooklm.google.com) with the same workshop account.
4. **Use the provided data:** modules that need uploads include a `Files/` folder. Only optional Module 6 needs live Gmail, Calendar, or Drive connectors.
5. **Follow cleanup steps:** Cloud labs can create billable resources. Delete or stop them when the lab or day guide tells you to.

Starting on Day 2? Follow the newcomer route in [Day 2](./DAY_2.md), then check the **Before you start** and **Data you need** sections in your chosen lab.

## How the Academy works

The modules build from document research to no-code agents, multi-step flows, code, and Cloud:

| Badge | Stage | Main tool |
|:---:|---|---|
| 🟢 | Setup and warm-up | Cloud Console and NotebookLM |
| 🟢 | Agent Designer: Chat pane | Build an agent by chatting |
| 🟡 | Agent Designer: Flow builder | Add Cloud data and visual steps |
| 🔴 | Multi-step agent crews | Coordinate focused subagents |
| 🔵 | Google-made agents | Run Deep Research and Idea Generation |
| 🟣 | Connectors and code | Use Workspace connectors or the Agent Development Kit (ADK) |
| ☁️ | Google Cloud | Store, run, protect, and deploy agents |

You do not need to complete every lab. Follow the day guide, then choose bonus labs that fit your pace and access.

Inside every lab:

1. Read the mission, prerequisites, and data requirements.
2. Work through the build steps.
3. Open the hints if you get stuck.
4. Open the full solution route when you need it.
5. Finish with the success check and cleanup instructions.
