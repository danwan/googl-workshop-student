# ☁️ Module 8 · Google Cloud Toolkit

> **Workshop role:** Day 2 core with Lab 8.1. Labs 8.2 to 8.5 are choice paths selected according to time and setup gates.

Your agents have to *live* somewhere. That "somewhere" is **Google Cloud**. In this module you learn the handful of Google Cloud moves that every agent-builder should know: store files, turn on a service, call it, work from the browser terminal, and then **ship a real agent to the web**.

> 🟣 **No connectors.** Everything here is the **Google Cloud Console** (and its built-in browser terminal). Just a browser.

## What you'll need

- A **dedicated billed Google Cloud workshop project for each participant**. Do not use a shared project for these labs: you create and delete real cloud resources.
- Use your dedicated billed workshop project with Project Owner/admin access. You enable the APIs and create, use, and delete the resources named in your chosen lab. Before starting, confirm the correct project is selected and **APIs & Services** opens. If either check fails, stop and ask the facilitator; do not switch projects.
- A **browser**. No local installs — the Console and **Cloud Shell** run entirely online.
- **Lab 8.2 includes its own deployable Formula starter**, and **Lab 8.3 Stage B includes its own inline Python/Vertex setup**. Reusing a Module 7 agent or environment is optional, not required.

## The labs

| Lab | Tier | What you do |
|---|---|---|
| **[Lab 8.1 — Cloud Foundations](./lab-8.1-cloud-foundations.md)** | 🟢 Foundations | Four 5-minute quests: create a **Cloud Storage** bucket, **enable an API**, **call it with zero code**, and run a command in **Cloud Shell**. |
| **[Lab 8.2 — Ship Your Agent](./lab-8.2-ship-your-agent.md)** | 🔴 Advanced | Deploy your Module 7 agent to **Agent Runtime** — it goes live as a managed, production-grade cloud backend. |
| **[Lab 8.3 — The Agent With Eyes](./lab-8.3-agent-with-eyes.md)** | 🔴 Advanced | Use the **Cloud Vision API** to give an agent sight — first no-code, then as a real ADK tool. |
| **[Lab 8.4 — The Document Detective](./lab-8.4-document-detective.md)** | 🔴 Advanced | Turn a messy TechBond **PDF** into structured data with **Document AI**. |
| **[Lab 8.5 — Shielding Your Agent](./lab-8.5-model-armor-prompt-security.md)** | 🔴 Advanced | Set up **Model Armor** to block prompt injections and mask PII. |

## Why it's here

Modules 1–7 taught you to *build* agents. This module teaches the **ground they stand on**: storage, APIs, the CLI, and deployment. It's the difference between a demo on your laptop and a service your whole team can use — the same "engineering agents to last" mindset from Module 7, one layer down. 🏗️
