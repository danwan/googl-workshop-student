# Module 0 · Prep-Work — Environment Setup 🛠️

> **Workshop role:** Required Day 1 preflight and self-service setup in your dedicated workshop project.

Welcome to **TechBond Agent Academy**! In this module, you'll set up your own **Gemini Enterprise** environment, enable **Agent Designer**, and provision your **Google Cloud Storage (GCS)** buckets. 

Your workshop account has Project Owner/admin access in its own dedicated billed Google Cloud project. You configure the environment yourself: enable APIs, create the Gemini Enterprise app, connect its data store, and retain the web app URL for later modules.
