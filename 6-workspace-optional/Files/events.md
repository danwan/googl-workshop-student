# Calendar Events for Workshop

Create these events in each attendee's Google Calendar. Schedule events relative to the workshop date to ensure scenarios work correctly.

## Scheduling Guide

- **[This Week]** = Events during the workshop week
- **[Next Week]** = Events in the week after workshop
- **[Recurring]** = Set up as recurring events
