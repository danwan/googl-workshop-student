# Module 2 · Level 1 — Agent Designer: Chat pane

> **Workshop role:** Day 1 core with Lab 2.1 and choice with Labs 2.2 to 2.3. Day 2 core for newcomers and choice for returning participants.

> **The big idea:** Build a working AI agent *just by chatting with it.* Type what you want, hand it a file, and it's alive. No connectors. No code. No setup headaches. 🎉

Welcome to the easiest on-ramp in the whole workshop. In **Agent Designer** (inside the Gemini Enterprise web app) there's a **Chat pane**. You describe the agent you want in plain English, optionally upload a file for it to learn from, hit **Submit**, and test it in **Preview**. Don't like something? You fix it by *chatting*, not coding.

## 🪄 How the chat pane works

1. **+ Create agent** → the chat pane opens.
2. **Type a prompt** describing the agent's job.
3. **Add files** (📎 upload icon) → hand it knowledge (a PDF, a doc).
4. **Submit** → Gemini builds it.
5. Test in the **Preview** tab → **refine by chatting** → **Create** to launch.

> ⚠️ **Connector-free zone.** The chat pane can't plug into Cloud Storage, BigQuery, or live Gmail. That's fine — we go connector-free by **uploading files** from this module's `Files/` folder. (Want agents that *do* things like send email? That's Module 6.)
