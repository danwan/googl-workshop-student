## 🏭 The story you're stepping into

You've just joined **TechBond Industries** — a company that makes super-strong industrial glue for cars, electronics, and factories. 🧪

TechBond is drowning in information: product spec sheets, customer emails, sales reports, quality incidents, packed calendars. Your job? **Build a team of AI agents to handle it all.** Every lab is a real TechBond problem you'll solve with an agent you build yourself.

## 🎮 How it works — level up

The Academy has **levels**, just like a video game. Start easy, earn confidence, then take on the boss challenges.

| Badge | Level | You'll use | What you'll build |
|:---:|---|---|---|
| 🟢 | **Prep-Work** | Cloud Console / Agent Designer | Enable Gemini Enterprise and set up your Cloud Storage bucket |
| 🟢 | **Warm-Up** | NotebookLM | Turn boring documents into podcasts, videos & mind maps |
| 🟢 | **Level 1** | Agent Designer (Chat pane) | Your first agent — built by *just chatting* |
| 🟡 | **Level 2** | Agent Designer (Flow builder) | A smarter agent that reads a whole document library |
| 🔴 | **Level 3** | Workflow Agent Pro | A team of agents that work together AND take action |
| 🔵 | **Explore** | Google-made power agents | Deep Research, Idea Generation & more — point and shoot |
| 🟣 | **Code and connectors** | Real connectors / Code | Live Gmail, Calendar, & Drive (Module 6), or build with code (Module 7) |
| ☁️ | **Cloud** | Google Cloud Console | Buckets, APIs & shipping your agent to the web (Module 8) |

👉 **You don't have to do every lab.** Pick the ones that excite you. Labs take 10–30 minutes; labs that need an app, cloud data, connectors, or earlier code say so at the top and include a catch-up route.

## 🚦 Before you start — the 3 things you need

1. **Your workshop account and project** 🔑 — sign in with the provided account and select your own dedicated billed Google Cloud project. Your account has Project Owner/admin access, so you enable APIs and create, configure, and delete the workshop resources yourself. Complete Module 0 before the agent-building labs.
2. **NotebookLM** 📓 — open [notebooklm.google.com](https://notebooklm.google.com) with the same account.
3. **The data files** 📦 — each module that needs uploads has a `Files/` folder with the TechBond PDFs for those labs. You don't need Gmail (except for the optional Module 6 connectors) or a database — you just upload files.

> **Confused by a word like "agent" or "Cloud Storage"?** Keep the **[Cheat Sheet](./CHEAT_SHEET.md)** open in another tab. It explains every term and shows exactly where to click.
